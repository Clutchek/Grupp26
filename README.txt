# Grupp26

Programmet är utvecklat i Netbeans.
För det första så är programmet främst designat för en 1080p 13 tums skärm som kör Windows.
Poyas mac kunde av någon anledning inte få färger i programmet.

För att få programmet att fungera:
1. Fixa .dat215 mappen
2. Öppna projektet i Netbeans.
3. Tryck "close" i "resolve issues" panelen som dyker upp.
4. Du kanske behöver lägga till en ny path till backenden.
5. Kolla att rätt version av Java används under libraries.
6. Starta programmet i Netbeans.
